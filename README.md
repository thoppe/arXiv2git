# arXiv2git
Links the [arXiv](http://arxiv.org/) to github repos

Why?

+ It's hard to search for papers like ["t \bar{t} W production and decay at NLO"](http://arxiv.org/abs/1204.5678)
+ It builds a database of links for other uses
